# 🚀 Terraform with AWS – 30 Days Learning Challenge

This repository documents my **30-day journey of learning Terraform with AWS**.

The goal of this challenge is to build a **strong foundation in Infrastructure as Code (IaC)** using **Terraform** while provisioning and managing infrastructure on **AWS**.

Each day contains a **separate folder** with notes, Terraform configurations, and practical examples demonstrating what was learned.

---

# 📌 Objective

The purpose of this repository is to:

- Learn **Terraform fundamentals**
- Understand **Infrastructure as Code (IaC)**
- Provision **AWS infrastructure using Terraform**
- Practice **real-world DevOps workflows**
- Document the learning process in a structured way
